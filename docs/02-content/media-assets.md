`/images/egor-photo.jpg`

## Open Graph

Рекомендуемое место:

`public/og/portfolio-cover.jpg`

`/og/portfolio-cover.jpg`

## Временное стоковое изображение

Первым используем временное стоковое изображение про обучение, ноутбук, программирование или наставничество. Важно, чтобы оно не выглядело слишком корпоративно и не спорило с дружелюбным наставническим тоном.

Подходящие варианты:

- Pexels: `A Tutor Teaching a Student` - https://www.pexels.com/photo/a-tutor-teaching-a-student-5311450/
- Pexels: `Focused tutor and student doing homework assignment using laptop` - https://www.pexels.com/photo/focused-tutor-and-student-doing-homework-assignment-using-laptop-4308095/
- Unsplash search: `Students with laptop` - https://unsplash.com/s/photos/students-with-laptop

Для hero/OG лучше брать горизонтальное изображение, где есть человек, ноутбук/тетради и ощущение спокойной учебной работы.

## Когда появится фото

- Использовать горизонтальный кадр для Open Graph.
- Использовать портретный или квадратный кадр для блока о себе.
- Не хранить исходники с большим весом в проекте без оптимизации.

## Минимальные размеры

- Фото для блока о себе: 800x800 или больше.
- Open Graph: 1200x630.
- Hero/background: 1600px по ширине или больше.
