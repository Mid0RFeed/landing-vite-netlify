# Собственный проект: аналог сборщика данных с авто Porsche

## Кратко

Собственный проект 2023 года: приложение и сайт для сбора, передачи и анализа данных с автомобиля в реальном времени.

## Задача

Создать приложение, которое передает информацию об авто в реальном времени, и сайт, где данные анализируются и становятся понятными обычному пользователю без поездки в сервис.

## Решение

Сбор данных с автомобиля, передача данных в реальном времени, аналитика на сайте, отображение истории ошибок и аномальных показателей.

## Период

- Год: 2023.
- Срок: 6 месяцев.
- Команда: 2 человека.

## Технологии

- Python.
- JS.
- Точный стек нужно уточнить.

## Результат

Обычный пользователь без поездки в сервис видит историю ошибок, аномальные показатели и состояние автомобиля в понятном интерфейсе.
