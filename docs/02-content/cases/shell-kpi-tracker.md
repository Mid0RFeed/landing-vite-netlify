# Shell: KPI-трекер

## Кратко

KPI-трекер для Shell, разработанный в 2020 году за 3 месяца командой из 3 человек.

## Задача

Сделать Android-приложение, которое помогает отслеживать KPI, упрощает мотивацию сотрудников и делает показатели понятнее для команды.

## Решение

Разработка приложения для Android, проработка логики KPI, пользовательских сценариев и интерфейса. Работа шла в небольшой команде из 3 человек.

## Период

- Год: 2020.
- Срок: 3 месяца.
- Команда: 3 человека.

## Технологии

- Android.
- Точный язык/стек нужно уточнить.

## Результат

Приложение повысило и упростило мотивацию сотрудников: KPI стали прозрачнее, а работа с показателями - удобнее.
