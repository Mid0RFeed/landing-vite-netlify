# МТС: приложение для управления умной техникой

## Кратко

Разработка приложения для управления умной техникой через Wi-Fi и Bluetooth в 2021 году за 6 месяцев.

## Задача

Создать приложение, через которое пользователь может управлять умной техникой по Wi-Fi и Bluetooth.

## Решение

Разработка приложения, проработка пользовательских сценариев управления устройствами, работа с Wi-Fi и Bluetooth-логикой.

## Период

- Год: 2021.
- Срок: 6 месяцев.
- Команда: 1 человек.

## Технологии

- Swift.
- JS.
- Wi-Fi.
- Bluetooth.
- Точный стек нужно уточнить.

## Результат

Пользователь получил приложение для управления умной техникой через беспроводные подключения.
