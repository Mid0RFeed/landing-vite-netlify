# Команды

## Разработка

```bash
npm run dev
```

## Сборка

```bash
npm run build
```

## Предпросмотр сборки

```bash
npm run preview
```

## Docker

```bash
docker compose up
```

## Частые проверки

- Сайт открывается локально.
- Нет ошибок в консоли браузера.
- Адаптивность проверена на мобильной ширине.
- `npm run build` проходит без ошибок.
