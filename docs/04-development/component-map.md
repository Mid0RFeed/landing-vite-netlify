# Карта компонентов

## Точка входа

- `src/App.tsx` - рендерит основной экран.
- `src/main.tsx` - React-точка входа.
- `index.html` - текущая статическая оболочка с секциями `hero`, `logos-intro`, `advantages`, `services`, `portfolio`, `contacts`.
- `src/hero-entry.tsx` - монтирует hero в `#hero-react-root`.
- `src/logos-entry.tsx` - монтирует логотипы в `#logos-react-root`.
- `src/advantages-entry.tsx` - монтирует преимущества в `#advantages-react-root`.
- `src/portfolio-entry.tsx` - монтирует портфолио в `#portfolio-react-root`.

## Demo-компоненты

- `src/components/demo.tsx` - hero.
- `src/components/gallery4-demo.tsx` - галерея / кейсы.
- `src/components/logos3-demo.tsx` - логотипы.
- `src/components/radial-orbital-timeline-demo.tsx` - таймлайн.

## Контент в коде

- Hero-тексты пока шаблонные и требуют замены.
- Логотипы компаний уже указаны в `src/components/logos3-demo.tsx`.
- Преимущества уже указаны в `src/components/radial-orbital-timeline-demo.tsx`.
- Кейсы уже указаны в `src/components/gallery4-demo.tsx`.
- Услуги и контакты сейчас находятся прямо в `index.html`.
