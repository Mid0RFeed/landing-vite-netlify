# Netlify checklist

## Перед деплоем

- [ ] `npm run build` проходит.
- [ ] Проверен `dist/`.
- [ ] Контакты актуальны.
- [ ] Мета-теги заполнены.
- [ ] Favicon отображается.
- [ ] Open Graph картинка задана.
- [ ] Мобильная версия проверена.

## Настройки Netlify

- Build command: `npm run build`
- Publish directory: `dist`

## После деплоя

- [ ] Открыть production URL.
- [ ] Проверить главную страницу.
- [ ] Проверить ссылки и CTA.
- [ ] Проверить Lighthouse.
