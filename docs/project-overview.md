# Обзор проекта

## Что это

Сайт-портфолио на Vite, React и Tailwind CSS. Проект подготовлен для локальной разработки, сборки и предпросмотра.

## Основные технологии

- Vite
- React
- TypeScript
- Tailwind CSS
- Radix UI
- Framer Motion

## Где смотреть код

- `src/App.tsx` - основной React-компонент.
- `src/components/` - компоненты интерфейса.
- `src/components/ui/` - переиспользуемые UI-компоненты.
- `src/index.css` и `src/style.css` - стили.
